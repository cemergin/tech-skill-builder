Use the `guide-creator` skill to create a comprehensive technical guide.

The guide-creator uses a diverge-converge workflow:
1. **Design** — brainstorm curriculum, define spiral structure, get approval
2. **Diverge** — dispatch 8-12 parallel agents to write chapters simultaneously
3. **Converge** — weave cross-references, ensure consistency, fill gaps
4. **Refine** — targeted agents on chapters that need deepening
5. **Polish** — final centralized pass

Start by invoking the guide-creator skill.
